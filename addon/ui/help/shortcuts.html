Lightning Tab Groups Keyboard Shortcuts

↑/↓Select a tab group.
Ctrl+↑/↓Move the selected tab group up or down.
←/→Hide/show tab group tabs.
EnterSwitch to the selected tab group.
1-9, a-zDirectly switch to the corresponding tab group.
InsertCreate a tab group.
Ctrl+InsertEdit the selected tab group.
DeleteDelete the selected tab group.
Ctrl+DeleteClose all tabs in the selected tab group.
Ctrl+zRestore the recently deleted tab group.
HomeMove the currently active tab to the selected tab group.
EndMove all or highlighted tabs to the selected tab group.
Shift+EndMove all tabs of the selected tab group to a new window.
Ctrl+End/HomeMove by switching to the selected tab group.
PageUpCopy the selected tab group to the clipboard.
Shift+PageUpCopy all tab groups to the clipboard.
PageDownPaste tab groups from the clipboard.
/Reload all tabs in the selected tab group.
=Search tabs.
F1Display help.
