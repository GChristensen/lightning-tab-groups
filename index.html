Lightning Tab Groups

Firefox Add-On (coming soon)

SEE ALSO: Enso Launcher, Scrapyard

Description

Lightning Tab Groups is a Firefox tab group manager based on tab hiding designed to be operated primarily from the keyboard. Some use cases require you to memorize a few simple keyboard shortcuts. In return, you get fast uncluttered user interface and the ability to instantly operate tabs. See the online help for a more detailed description of the add-on features.

Key Features

  • Switching between tab groups with keyboard shortcuts.
  • Use of color tab group mnemonics.
  • Use of emoji as a tab group mnemonic.
  • Ability to close or reload all tabs in the tab group.
  • Ability to copy and paste tab groups between browser instances.
  • iShell integration.
